About Project

This app is a project created for showing the analysis of Seattle Public Library dataset from several tools, which are Weka, Tensorflow, and Spark.

Original dataset can be accessed through https://www.kaggle.com/seattle-public-library/seattle-library-checkout-records

Datasets were preprocessed using tools such as Hue and custom Java application. For the Weka Statistics, records from the datasets were cut down from the original number to 10,000 per year to match the capability of Weka.
